There is very clearly a difference between the characteristics depending on
which mode domiantes. When ACC dominates, $\Delta$ Smax tends to be much
smaller. Put in other terms, when ACC dominates, the activation characteristics
of the full aerosol mixture tend to closely follow that when only ACC is
present. This is true in either maritime or continental regimes.

To illustrate this point in more detail, we can compare the distributions of
Smax for each region:

